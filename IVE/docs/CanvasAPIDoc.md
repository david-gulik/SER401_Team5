# canvasapi
## Introduction

canvasapi is an object-oriented interface for the Canvas REST API.
It provides elegant and intuitive Python methods as a substitute for JSON/HTML requests to the REST API. Each canvasapi 
method corresponds to a REST request and includes authentication, pagination, and rate limits. We will use the canvasapi
in our code to access gradebook data from Canvas.

## Setup

1. Install canvasapi:  

`pip install canvasapi`

2. Generate API token from Canvas
3. Import canvasapi into your project:  

`from canvasapi import Canvas`


## Objects
canvaspi translates JSON objects returned by the Canvas REST API into Python objects. 
### Canvas, itself
To instantiate Canvas in your project:  

`canvas = canvas.Canvas(api_url, api_key)`  

Where `api_url` is your Canvas URL (in our case, https://canvas.asu.edu) and `api_key` is your API token generated by Canvas.
### Courses
Any course in the Canvas API is represented by a Course object, and any Course for which you have instructor-level permissions 
may be gathered using its Course ID:  

`course = canvas.get_course(courseNum)`  

| Attribute             | Type | Description  |
|:----------------------|:-----|:-------------|
| `id`                  | `int ` | Course ID    |
| `name`                |`str`|Course Name| 
| `course_code`         |`str`|Course Code, e.g., "SER401"|
| `start_at`            |`str`|Start Date|
| `end_at`              |`str`|End Date|
| `grading_standard_id` |`int`|Grading Standard ID|

### Enrollments
Any student or TA in any Course is represented by an Enrollment in the Course:

`enrollments = course.getEnrollments()`  

To filter out test student/TA enrollments, we can use a keyword argument in our function call:  

`students = course.getEnrollments(type="StudentEnrollment")`


| Attribute           | Type                   | Description                                                                |
|:--------------------|:-----------------------|:---------------------------------------------------------------------------|
| `id`                | `int `                 | Enrollment ID                                                              |
| `user_id`           | `int`                  | Canvas user ID                                                             | 
| `user`              | `dict[str, Any]`       | User Info (name, login, etc.)                                              |
| `type`              | `str`                  | Enrollment Type, e.g., "StudentEnrollment", "TAEnrollment"                 |
| `enrollment_status` | `str`                  | `"active"`,`"completed"`,`"inactive"`                                      |
| `grades`            | `dict[str, float/str]` | Contains `"current_score"`,`"final_score"`,`"current_grade"`,`"final_grade"` |

### Users
We can retrieve further user information in the User dictionary of any Enrollment:  

| Attribute       | Type   | Description                         |
|:----------------|:-------|:------------------------------------|
| `id`            | `int ` | User ID                             |
| `name`          | `str`  | Name in "First Last" format         | 
| `sortable_name` | `str`  | Name in "Last, First" format        |
| `short_name`    | `str`  | Shortened name for display purposes |
| `login_id`      | `str`  | Canvas login username               |
| `email`         | `str`  | Primary email address               |


### Assignments
What would a Course be without Assignments?? We can access them all at once:  

`assignments = course.getAssignments()`  
  
Or get a single assignment:

`assignment = course.getAssignment(assignmentNum)`  


| Attribute             | Type    | Description             |
|:----------------------|:--------|:------------------------|
| `id`                  | `int `  | Assignment ID           |
| `name`                | `str`   | Assignment Name         | 
| `description`         | `str`   | HTML Description        |
| `points_possible`     | `float` | Maximum Points Possible |
| `assignment_group_id` | `int`   | Assignee Group ID       |
| `due_at`              | `str`   | Due Date                |
|`unlock_at`|`str`|Start of Assignment Availability|
|`lock_at`|`str`|End of Assignment Availability|
|`grading_type`|`str`|`"points"`,`"letter_grade"`,etc.|
|`published`|`bool`|Can Students See It?|
|`muted`|`bool`|Are Grades [Not] Visible?|
|`submission_types`|`list[str]`|`"online_upload"`,`"on_paper"`, etc.|

### Submissions

When a student/TA/etc. submits to an Assignment, a Submission is created. Similar to Assignments, we can grab them all:

`submissions = assignment.getSubmissions()`

Or get a single student's submission:

`submission = assignment.getSubmissions(studentID)`


| Attribute       | Type    | Description                 |
|:----------------|:--------|:----------------------------|
| `id`            | `int `  | Submission ID               |
| `assignment_id` | `int`   | Corresponding Assignment ID | 
| `user_id`       | `int`   | Submitter's ID              |
| `score`         | `float` | Points Awarded              |
| `submitted_at`  | `str`   | Timestamp of Submission     |
| `graded_at`     | `str`   | Timestamp of Grading        |
| `grader_id` | `int` | Grader's ID |
|`workflow_state`|`str`|`"graded"`,`"submitted"`,`"unsubmitted"`|
|`late`|`bool`|Was it submitted late?|
|`missing`|`bool`|Is it missing?|
|`excused`|`bool`|Is the student excused from this assignment?|
|`attempt`|`int`|Attempt no. #|
